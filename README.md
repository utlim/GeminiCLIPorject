# 📚 YES24 vs 교보문고 컴퓨터/IT 베스트셀러 데이터 분석 & BI 대시보드

본 프로젝트는 국내 양대 서점인 **YES24(1,000위)**와 **교보문고(100위)**의 컴퓨터/IT 분야 베스트셀러 도서 데이터를 크로스 플랫폼 관점에서 심층적으로 비교 분석하고, 이를 시각적으로 탐색할 수 있는 **반응형 웹 대시보드(BI Dashboard)** 구축 프로젝트입니다.

## 🎨 웹 대시보드 주요 핵심 기능

### 1. 3대 전역 디자인 테마 (Theme Switcher)
- **Sleek Dark Bento Theme**: 심해 블루톤의 차분하고 입체적인 벤토 카드 격자 UI.
- **Glassmorphism Theme**: 오로라 파스텔 그라디언트 백그라운드와 반투명 글래스 레이아웃의 라이트 모드.
- **Neon Cyberpunk Theme**: 칠흑의 어둠 속에서 형광 시안과 핑크 네온 라인이 빛나는 글로우 UI.
- 테마 전환 시 **Chart.js의 폰트 및 그리드 색상도 실시간 동기화** 갱신됩니다.

### 2. 5대 핵심 Chart.js 시각화
- **가격 분포 비교**: 두 서점의 도서 가격대 구간별 점유 비교 (Multi-series Bar)
- **출판사 점유율/평점**: 메이저 출판사(길벗, 한빛미디어 등 6개사)의 점유 교차 비교 (Radar)
- **평점 만족도 비교**: 인플레이션된 도서 평점의 극단적 만족도 쏠림 분석 (Doughnut)
- **순위별 성과 비교**: 순위 하락에 따른 리뷰 수 및 판매지수 성과 분석 (Combo Bar-Line)
- **TF-IDF 핵심 키워드**: 도서명 빈출 키워드의 서점별 가중치 비교 (Horizontal Grouped Bar)

### 3. 6대 대화형 데이터 탐색 (Interactive Features)
- **인메모리 고속 검색/정렬**: 런타임 지연 없는 도서명, 저자, 출판사 통합 검색 및 정렬.
- **1:1 도서 교차 매칭**: 도서명 불필요 특수문자 정규화 및 Fuzzy Matching 알고리즘 기반 1:1 도서 매핑 비교 카드 제공 (순위 편차, 실구매가 차액, 평점/리뷰 수 나란히 오버랩 비교).
- **가격 범위 슬라이더**: 양방향 레인지 슬라이더를 조절하여 예산 범위 내 도서들로 실시간 대시보드 전체 리프레시.
- **출판사/할인율 필터**: 체크박스를 통한 서브 그룹 맞춤 대시보드 뷰 제공.
- **트렌드 단축 태그 필터**: `#AI`, `#클로드`, `#시나공` 등 핫 키워드 클릭 시 관련 도서 즉시 연동.
- **실시간 요약 통계 테이블**: 필터 조건 변동 시 데이터 수량, 평균 정가, 평균 판매가, 평균 평점, 누적 리뷰 수가 실시간으로 재계산되어 통계 결과 업데이트.

## 🚀 로컬 실행 및 데이터 업데이트 가이드

### 1. 의존성 패키지 설치
로컬에서 프로젝트를 구동하기 위해 node 패키지를 설치합니다.
```bash
cd dashboard
npm install
```

### 2. 로컬 개발 서버 구동
Vite 개발 서버를 띄워 로컬 브라우저에서 구동합니다.
```bash
npm run dev
# 기본 접속 주소: http://localhost:5173/
```

### 3. 데이터 파이프라인 가동 (JSON 업데이트)
YES24 및 교보문고의 원본 CSV 데이터를 대시보드가 사용하는 JSON 형식으로 정적 컴파일 업데이트하려면 아래 파이썬 스크립트를 구동합니다.
```bash
# 워크스페이스 루트에서 실행
python dashboard/src/convert_data.py
```
